# Conformance

This document tracks the Go SDK's coverage of the spec at
[`../spec/docs/draft-arcp-1.1.md`](../spec/docs/draft-arcp-1.1.md).
The matrix is generated by the harness at
[`tests/conformance/`](./tests/conformance/); run it with:

```sh
go test ./tests/conformance/...
```

Set `ARCP_CONFORMANCE_OUT=/path/to/conformance.json` to emit the
machine-readable summary.
