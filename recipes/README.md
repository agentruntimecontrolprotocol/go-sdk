# Recipes

These runnable recipes mirror the TypeScript SDK scenarios using Go
client, server, transport, and middleware packages.

| Recipe | Entry points |
| --- | --- |
| `email-vendor-leases` | `go run ./recipes/email-vendor-leases/server` and `go run ./recipes/email-vendor-leases/client` |
| `mcp-skill` | `go run ./recipes/mcp-skill/server` |
| `multi-agent-budget` | `go run ./recipes/multi-agent-budget/server` and `go run ./recipes/multi-agent-budget/client` |
| `stream-resume` | `go run ./recipes/stream-resume/server` and `go run ./recipes/stream-resume/client` |
